## GitHub Pages

1. Создай новый Public репозиторий.
2. Загрузи все файлы из этой папки: `index.html`, `style.css`, `script.js`, `images`.
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Нажми Save.
